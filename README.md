# 2. How To

Install Conda (https://docs.anaconda.com/miniconda/)
Activate Conda (Linux eval "$(/home/malisha/miniforge3/bin/conda shell.bash hook)")

Build project

* Windows
    ```
    run.bat build
    ```
* Linux
    ```
    bash run.sh build
    ```

Run streamlit UI

```
run.bat streamlit
```

Get help

```
run.bat build --help
```
